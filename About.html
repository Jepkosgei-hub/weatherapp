Welcome to Our Weather App - Accurate, Real-Time Forecasts for Kenyan Cities!

About This Weather App

The Weather App is a simple, user-friendly platform designed to provide up-to-date weather forecasts for various cities across Kenya. Whether you're planning a trip or just curious about local weather conditions, this app offers real-time temperature, humidity, and weather condition updates.

Features

  • 🌦 7-day weather forecast for cities like Nairobi, Mombasa, Nakuru, and Eldoret
  • 📍 Interactive weather map using Leaflet and OpenWeatherMap
  • 🔍 Search functionality to get forecast for any city
  • 🌡 Weather details include temperature, humidity, and condition
  • 🚀 Fast and responsive interface

Technologies Used

  • 📡 OpenWeatherMap API - for real-time weather data
  • 🗺 Leaflet.js - for the interactive map display
  • 🧑‍💻 HTML5, CSS3, JavaScript - for front-end development
  • 🧭 OpenStreetMap - as the base map layer

Purpose

This project was created to help users in Kenya easily access reliable and current weather forecasts. It's also a practical demonstration of integrating public APIs with front-end web development tools.
